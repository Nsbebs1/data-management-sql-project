# Data Management SQL Project

A SQL project simulating data sources within a data management platform.

## Project Idea
This project represents data sources such as APIs, databases, and open data portals.  
It includes SQL queries to analyze source types, departments, activity status, update dates, and data quality scores.

## Skills Applied
- SQL
- Database Design
- Data Management
- Data Quality Analysis
- Data Source Monitoring

## Tables
- DataSources

## Queries Included
- Count total data sources
- Filter active and inactive sources
- Analyze data sources by type
- Calculate average data quality score
- Identify low-quality data sources
- Sort sources by latest update
